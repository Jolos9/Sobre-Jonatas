Le site Web se termine enfin

Le 14/11/2023, j'ai créé mon premier site Web en HTML. Il vise à informer les principaux sujets me concernant pour vous aider avec d'éventuelles opportunités d'emploi futurs.

Plus pour l'instant, créez-vous un site Web ?

Il y a plusieurs raisons pour lesquelles j'ai créé ce site. Certains disent que c'était pour m'aider professionnellement à l'avenir, d'autres que c'était pour que je puisse posséder une véritable expérience HTML, et ils n'ont pas entièrement tort. Cependant, la véritable raison pour laquelle j'ai créé ce site était pour l'activité du professeur Evandro, MTC.
